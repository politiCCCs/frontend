## Development

First, install all the dependencies:

```shell
npm install
```

To run the app in development mode, run:

```shell
npm start
```

This opens the app on [http://localhost:3000](http://localhost:3000) in your browser.

## Building

To build the app, run:

```shell
npm run build
```
